gap（行・列の間隔）のデモ一覧

gap / row-gap / column-gap の挙動を視覚的にまとめたデモです。
行・列がどのように離れていくか、また gap を 0 にするとどう変わるかを確認できます。

1. 行・列共通の gap（gap: 12px）

最も基本的な gap。行と列が同じ間隔で開く。

.grid-gap-12 {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px;
}

2. 行と列で異なる gap（row-gap: 20px; column-gap: 40px;）

行の間は20px、列の間は40px。
