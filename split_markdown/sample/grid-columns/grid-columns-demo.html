grid-template-columns の基本例

代表的な列幅の指定パターンをまとめたデモです。
各例で grid-template-columns の指定を変更してレイアウトの違いを確認できます。

1. 3列を等分する（1fr 1fr 1fr）

横幅を3等分する基本的なパターン。画面幅に応じて各列が同じ割合で伸び縮みします。

1
2
3
.grid-equal {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 12px;
}

2. 固定幅 + 可変幅（200px 1fr）

左のカラムは常に200px、右のカラムが残りの幅をすべて受け取ります。 サイドバー＋コンテンツのようなレイアウトに使います。

サイドバー
メインコンテンツ
.grid-fixed-flex {
  display: grid;
  grid-template-columns: 200px 1fr;
  gap: 12px;
}

3. 全部固定幅（150px 150px 150px）

すべての列を同じ固定幅にするパターン。カード一覧やアイコンのグリッドなど、 幅を揃えたいときに使用します。

カード1
カード2
カード3
.grid-fixed-all {
  display: grid;
  grid-template-columns: 150px 150px 150px;
  gap: 12px;
}

4. 自然幅 + 可変幅（auto 1fr）

左のカラムはコンテンツの幅に合わせて決まり、右のカラムが残りのスペースを受け取ります。 ラベル + 入力欄のようなレイアウトに向いています。

ラベルのテキスト
入力欄や本文テキスト
.grid-auto-fr {
  display: grid;
  grid-template-columns: auto 1fr;
  gap: 12px;
}

5. 最小〜最大の範囲で伸縮（minmax(150px, 1fr)）

列幅の下限と上限を指定するパターン。小さい画面では潰れすぎないようにしつつ、 大きい画面では余裕を持って広がるレイアウトにできます。

左カラム
右カラム
.grid-minmax {
  display: grid;
  grid-template-columns: minmax(150px, 1fr) 1fr;
  gap: 12px;
}
