grid-template-columns の例一覧（完全版）

代表的な列幅指定のパターンをすべてまとめて視覚的に確認できるデモです。
全部使う必要はありませんが、Grid の特性上「どんな指定があるか」を見ておくことが理解の近道になります。

1. 3列を等分する（1fr 1fr 1fr）

横幅を均等に3分割する基本パターン。

1
2
3
.grid-equal {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 12px;
}

2. 固定幅＋可変幅（200px 1fr）

左は固定幅、右が残りすべてを受け取る。

サイドバー
メイン
.grid-fixed-flex {
  display: grid;
  grid-template-columns: 200px 1fr;
  gap: 12px;
}

3. 全部固定幅（150px 150px 150px）

カード一覧など幅を揃えたい場面に。

A
B
C
.grid-fixed-all {
  display: grid;
  grid-template-columns: 150px 150px 150px;
  gap: 12px;
}

4. 自然幅＋可変幅（auto 1fr）

左列は内容に合わせる、右列は残りすべて。

ラベルテキスト
本文（長めの文章にも耐える）
.grid-auto-fr {
  display: grid;
  grid-template-columns: auto 1fr;
  gap: 12px;
}

5. 最小〜最大で伸縮（minmax(150px, 1fr) 1fr）

潰れすぎない＋広がる可変レイアウト。

左
右
.grid-minmax {
  display: grid;
  grid-template-columns: minmax(150px, 1fr) 1fr;
  gap: 12px;
}

6. repeat（repeat(3, 1fr)）

等分3列の省略形。

1
2
3
.grid-repeat {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px;
}

7. auto-fill（repeat(auto-fill, minmax(150px, 1fr))）

スペースがあれば「空の枠」も含めて横に詰めて配置する。 画面幅に応じて列数が自動的に増える。

1
2
3
4
.grid-autofill {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
  gap: 12px;
}

8. auto-fit（repeat(auto-fit, minmax(150px, 1fr))）

空きスペースを埋めるように要素を「伸ばす」。 auto-fill と違って余白を持たせずフィットさせる。

1
2
3
4
.grid-autofit {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
  gap: 12px;
}
