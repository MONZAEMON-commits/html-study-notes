grid-template-rows の基本例

行の高さ（Row）に関する代表的な指定パターンをまとめたデモです。
各例で grid-template-rows の違いを視覚的に確認できます。

1. 固定行（80px auto 80px）

上と下が固定高さ、中央が可変。

Header
Main
Footer
.grid-fixed-header {
  display: grid;
  height: 240px;
  grid-template-rows: 80px auto 80px;
  gap: 12px;
}

2. 等分3行（1fr 1fr 1fr）

高さを均等に3分割する。

Row1
Row2
Row3
.grid-equal-rows {
  display: grid;
  height: 300px;
  grid-template-rows: 1fr 1fr 1fr;
  gap: 12px;
}

3. 全部固定高さ（100px 100px 100px）

どの行も必ず同じ高さ。

Row1
Row2
Row3
.grid-fixed-all {
  display: grid;
  grid-template-rows: 100px 100px 100px;
  gap: 12px;
}

4. 自然高さ + 可変高さ（auto 1fr）

上はコンテンツの高さ、下は残りすべて。

自然高さ（テキスト量に応じて決まる）
残り全部の高さ
.grid-auto-fr {
  display: grid;
  height: 260px;
  grid-template-rows: auto 1fr;
  gap: 12px;
}

5. minmax（minmax(120px, auto) 1fr）

最小値を下回らず、必要に応じて伸びる。

Row1
Row2
.grid-minmax {
  display: grid;
  height: 260px;
  grid-template-rows: minmax(120px, auto) 1fr;
  gap: 12px;
}

6. repeat（repeat(4, 1fr)）

同じ行を複数生成したい場合に便利。

Row1
Row2
Row3
Row4
.grid-repeat {
  display: grid;
  height: 240px;
  grid-template-rows: repeat(4, 1fr);
  gap: 12px;
}
