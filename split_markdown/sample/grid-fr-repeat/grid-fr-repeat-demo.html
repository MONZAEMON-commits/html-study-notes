fr / repeat() のデモ一覧

fr（比率配分）と repeat()（繰り返し生成）の挙動をまとめて視覚的に確認できるデモです。
auto-fill と auto-fit の違いも含め、Grid の柔軟性が分かるよう構成されています。

1. 等分（1fr 1fr 1fr）

残り幅を均等に3つへ分配する最も基本的な fr。

1
2
3
.demo-fr-equal {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 12px;
}

2. 比率（2fr 1fr）

左が 2、右が 1 の比率で残り幅を分配。

左（2fr）
右（1fr）
.demo-fr-2-1 {
  display: grid;
  grid-template-columns: 2fr 1fr;
  gap: 12px;
}

3. センター強調（1fr 2fr 1fr）

中央を強調したいときによく使う。

L
C
R
.demo-fr-center {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 12px;
}

4. repeat（repeat(3, 1fr)）

3等分を簡潔に書く省略記法。

A
B
C
.demo-repeat-3fr {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px;
}

5. 固定幅の繰り返し（repeat(4, 150px)）

幅が揃った固定グリッドを作る。

1
2
3
4
.demo-repeat-fixed {
  display: grid;
  grid-template-columns: repeat(4, 150px);
  gap: 12px;
}

6. 自然幅の繰り返し（repeat(3, auto)）

auto が必要幅を取り、行が自然に広がる。

短文
長めの文章が入ります
文字量の差を確認
.demo-repeat-auto {
  display: grid;
  grid-template-columns: repeat(3, auto);
  gap: 12px;
}

7. 柔軟なカラム群（repeat(5, minmax(100px, 1fr))）

小さい画面では100pxを下回らず、大画面では広がる。

8. auto-fill（repeat(auto-fill, minmax(150px, 1fr))）
