auto-fill / auto-fit のデモ一覧

auto-fill と auto-fit の違いを視覚的に比較できるデモです。
ウィンドウ幅の変化や、カードの数が少ない場合の挙動が最もわかりやすい構成になっています。

1. auto-fill（空セルを残す）（repeat(auto-fill, minmax(150px, 1fr))）

可能な限りカラムを詰めて生成するが、空き枠（空セル）はそのまま残る。

.demo-autofill {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
  gap: 12px;
}

2. auto-fit（空き吸収で伸びる）（repeat(auto-fit, minmax(150px, 1fr))）

空セルが“潰れて”見えなくなり、残った要素が横に伸びてフィットする。

.demo-autofit {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
  gap: 12px;
}
