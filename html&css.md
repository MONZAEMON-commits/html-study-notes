# HTML/CSS webデザイン
## HTMLの基本
- 基本の書き方<br>
    <タグ名>～</タグ名>
- 属性と属性値について<br>
<タグ名 付加情報=”内容”>～</タグ名>

- HTMLの骨格の作り方(お決まりセット)<br>
    VSC上で「!」を入力してEnterで以下に変換される。

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
- 注意事項
- 本文以外はすべて半角英数字
- タグの互い違いに注意

-お決まりセットの構造
| タグ                                                                     | 役割                             | 詳細な説明                                                                                                                                                                                                                                           |
| ------------------------------------------------------------------------ | -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `<!DOCTYPE html>`                                                        | HTML文書のバージョン宣言         | ブラウザに「このページはHTML5で書かれています」と知らせるための宣言。古いHTML（例：HTML4やXHTML）では別の宣言が必要だったが、HTML5ではこれだけでOK。これがないとブラウザが「互換モード」で表示してしまい、意図しないデザイン崩れを起こす場合がある。 |
| `<html lang="en">`                                                       | HTML全体のルート要素             | ページ全体を囲む最上位のタグ。すべての要素はこの中に含まれる。`lang`属性はページの言語を指定し、音声読み上げソフトや検索エンジンが正しく処理するために使われる。日本語サイトなら `lang="ja"` にするのが一般的。                                      |
| `<head>`                                                                 | ページの設定情報を記述する部分   | ページの表示内容ではなく、**ブラウザや検索エンジンに伝える情報**をまとめる領域。文字コード、タイトル、外部CSSやJavaScriptの読み込み、SEO情報、SNS連携用のメタ情報（OGPなど）をここに記述する。                                                       |
| `<meta charset="UTF-8">`                                                 | 文字コードの指定                 | ページで使う文字のエンコーディングを「UTF-8（ユニコード）」に指定。UTF-8は世界中の文字を扱えるため、国際標準的に推奨される。これがないと文字化けが起こる可能性がある。                                                                               |
| `<meta name="viewport" content="width=device-width, initial-scale=1.0">` | スマホ・タブレット向けの表示設定 | 表示領域（viewport）をデバイス幅に合わせる設定。`width=device-width` でスマホの画面幅に合わせ、`initial-scale=1.0` で初期の拡大率を1倍にする。これがないと、スマホでPCサイズのページが縮小されて表示されてしまう。レスポンシブ対応サイトでは必須。   |
| `<title>Document</title>`                                                | ページタイトル                   | ブラウザのタブ名や検索結果、ブックマークに表示されるページ名。SEO（検索エンジン最適化）的にも非常に重要で、ページ内容を端的に表すタイトルを設定するのが基本。例：`<title>社員体調管理システム｜ポートフォリオ</title>`                               |
| `<body>`                                                                 | 実際に表示されるコンテンツ       | ページ上にユーザーが目にする要素をすべて記述する部分。見出し、段落、画像、リンク、ボタン、フォームなどすべてのUI要素はこの中に書く。CSSやJavaScriptを使ってデザインや動きを加えるのもここで行う。                                                    |
| `</html>`                                                                | HTML文書の終端                   | ページ全体の終了を明示するタグ。`<html>`の閉じタグであり、HTML構造を正しく完結させる役割を持つ。閉じ忘れるとHTML解析エラーが発生することがある。                                                                                                     |

### 🧩 href 属性の主な指定パターン

| 種類                           | 記述例                                              | 説明                                                          |
| ------------------------------ | --------------------------------------------------- | ------------------------------------------------------------- |
| **絶対URL**                    | `<a href="https://example.com/">サイトへ</a>`       | 完全なURLを指定。外部サイトへリンクするときに使用。           |
| **相対URL（同一サイト内）**    | `<a href="about.html">会社概要</a>`                 | 現在のページを基準にしたパス。サイト内リンクで使う。          |
| **ルート相対パス**             | `<a href="/images/pic.png">画像</a>`                | ドメイン直下を基準に指定。サイト構造が深くても安定。          |
| **ページ内リンク（アンカー）** | `<a href="#section1">詳しく見る</a>`                | 同じページ内のid属性（`id="section1"`）へジャンプ。           |
| **メールリンク**               | `<a href="mailto:info@example.com">メール送信</a>`  | メールソフトを開いて送信先を指定。                            |
| **電話リンク**                 | `<a href="tel:09012345678">電話する</a>`            | スマホでタップしたときに電話発信する。                        |
| **JavaScriptリンク（非推奨）** | `<a href="javascript:void(0)">クリック</a>`         | クリックイベント用。ただし現在は`onclick`などを使うのが主流。 |
| **ファイルダウンロード**       | `<a href="data/file.zip" download>ダウンロード</a>` | `download`属性を追加で使うと、直接DLできる。                  |
| **空リンク（仮置き）**         | `<a href="#">リンク</a>`                            | 仮のリンク。クリックでページ上部に戻る。開発中に一時利用。    |

### 基本構文
```css
セレクタ {
  プロパティ: 値;
}
```
🧩 セレクタ（selector）とは？<br>
CSSが「どのHTML要素にスタイルを当てるか」を決める指定のこと。<br>
| 種類           | 書き方       | 対象                        | 例                            |
| :------------- | :----------- | :-------------------------- | :---------------------------- |
| 要素セレクタ   | `p`          | 特定のタグ                  | `<p>テキスト</p>`             |
| クラスセレクタ | `.name`      | `class="name"` が付いた要素 | `<p class="name">...</p>`     |
| IDセレクタ     | `#main`      | `id="main"` が付いた要素    | `<div id="main">...</div>`    |
| 子孫セレクタ   | `div p`      | div内のp要素                | `<div><p>...</p></div>`       |
| グループ化     | `h1, h2, h3` | 複数要素をまとめて指定      | `h1, h2, h3 { color: blue; }` |

⚙️プロパティ（property）と値（value）<br>
スタイルの「項目」と「設定内容」をペアで指定します。<br>
| プロパティ           | 値        | 意味               |
| :------------------- | :-------- | :----------------- |
| `color`              | `red`     | 文字色             |
| `background-color`   | `skyblue` | 背景色             |
| `font-size`          | `16px`    | 文字サイズ         |
| `width` / `height`   | `200px`   | 幅・高さ           |
| `margin` / `padding` | `10px`    | 余白（外側／内側） |

🗂️ CSSの書き込み場所（3パターン）<br>
| 方法                    | 書き方                                     | 特徴                                          |
| :---------------------- | :----------------------------------------- | :-------------------------------------------- |
| **外部CSS**（おすすめ） | `<link rel="stylesheet" href="style.css">` | HTMLとCSSを分離できる。複数ページで共有可能。 |
| **内部CSS**             | `<style>p { color: red; }</style>`         | 1ページ内で完結。テストや学習用に便利。       |
| **インラインCSS**       | `<p style="color:red;">テキスト</p>`       | 一部の要素だけ変更。基本は避ける。            |

まとめ
| 要素       | 意味               | 例                             |
| :--------- | :----------------- | :----------------------------- |
| セレクタ   | どのHTMLに当てるか | `p`, `.class`, `#id`           |
| プロパティ | 何を変えるか       | `color`, `font-size`, `margin` |
| 値         | どう変えるか       | `red`, `16px`, `auto`          |
| コメント   | メモ書き           | `/* これはコメント */`         |

CSSの基本ルール：「どれが勝つか」は決まっている<br>
同じ要素に複数のスタイルが当たるとき、<br>
CSSは次の順で「どれを適用するか」を判断します<br>

| 優先順位                               | 条件                                    | 例                         |
| :------------------------------------- | :-------------------------------------- | :------------------------- |
| **① インラインスタイル**               | HTMLタグ内に直接書かれたCSS             | `<p style="color:red;">`   |
| **② IDセレクタ**                       | `#id名` で指定されたスタイル            | `#main { color:blue; }`    |
| **③ クラス／属性／疑似クラスセレクタ** | `.class名`, `[type=text]`, `:hover`など | `.notice { color:green; }` |
| **④ 要素セレクタ**                     | `p`, `h1`, `div` など                   | `p { color:gray; }`        |
| **⑤ *（全称セレクタ）**                | すべての要素に当たる                    | `* { color:black; }`       |

### ブロックボックスとインラインボックスについて
📦 1. すべての要素は「ボックス」でできている。<br>
HTMLの要素はすべて「ボックス」として描画されます。<br>
そのボックスには大きく2種類あります👇
| 種類                   | 英語       | 主な用途                                     |
| :--------------------- | :--------- | :------------------------------------------- |
| **ブロックボックス**   | Block Box  | 段落・セクションのように**縦方向に並ぶ要素** |
| **インラインボックス** | Inline Box | 文章中に**横方向に並ぶ要素**                 |

🧱 2. ブロックボックス（Block Box）<br>
▶ 特徴
- 縦方向に積み重なる（次の要素は改行される）
- 幅は親要素いっぱい（デフォルトで width: 100%）
- margin や padding が上下左右すべて有効
- width, height を指定できる<br>

▶ 代表的なタグ
`<div>, <p>, <h1>～<h6>, <section>, <ul>, <ol>, <table>` など<br>
🧩 3. インラインボックス（Inline Box）<br>
▶ 特徴
- 横方向に並ぶ（文章の流れに沿う）
- width や height は指定しても効かない（文字サイズや内容で決まる）
- margin や padding の上下は効かない（左右のみ有効）
- 改行を入れない限り、同じ行に並ぶ
▶ 代表的なタグ
`<span>, <a>, <strong>, <em>, <b>, <i>, <img>` など
